linstor.protobuf_to_dict package¶

Module contents¶

A small Python library for creating dicts from protocol buffers. Useful as an intermediate step before serialization (e.g. to JSON).

This library is used, as older protobuf versions don’t have a JSON serializer yet.

linstor.protobuf_to_dict.dict_to_protobuf(pb_klass_or_instance, values, type_callable_map={12: <function get_bytes>}, strict=True)¶

Populates a protobuf model from a dictionary.

Parameters:
  • pb_klass_or_instance (a type or instance of a subclass of google.protobuf.message.Message) – a protobuf message class, or an protobuf instance
  • values (dict) – a dictionary of values. Repeated and nested values are fully supported.
  • type_callable_map (dict) – a mapping of protobuf types to callables for setting values on the target instance.
  • strict (bool) – complain if keys in the map are not fields on the message.
